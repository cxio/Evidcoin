# Proposal: Comparison Instructions [104–111]（提案：比较指令）

> **Prerequisite:** 本规范基于 [Base Constraints](0.Base-Constraints.md) 文档，后者定义了贯穿全文的指令格式、运行时模型与符号约定。


## 1. Overview（概述）

比较指令占用 opcode 范围 **[104–111]**（共 8 条）。它们用于比较两个或多个值，并返回布尔结果。这类指令是生成布尔条件的主要机制，常用于接入 `PASS`、`CHECK`、`IF{}` 及其他控制流指令。

### Float Comparison Caveat（Float 比较注意事项）

对 Float 值的相等比较（`EQUAL`）在技术上是支持的，但由于浮点精度问题，应 **used with caution**。如需可靠的 Float 比较，建议改用工具指令 `COMFLO`，该指令支持基于 epsilon 的近似相等。


## 2. Instruction Definitions（指令定义）

### 2.1 Relational Comparisons（关系比较）

```
104     EQUAL       Equal (a == b)
105     NEQUAL      Not equal (a != b)
106     LT          Less than (a < b)
107     LTE         Less than or equal (a <= b)
108     GT          Greater than (a > b)
109     GTE         Greater than or equal (a >= b)
```

- **Arguments:** 两个可比较值。
- **Return value:** Bool（`true` 或 `false`）。
- **Supported types:**
  - 所有数值类型：Byte、Rune、Int、BigInt、Float
  - String（按字典序比较）
  - Bytes（按从左到右逐字节比较）
  - Bool：仅 `EQUAL` 与 `NEQUAL` 支持（布尔值不定义大小顺序）
- **Cross-type comparison:** 当比较不同数值类型时，应用标准提升规则（Byte → Rune → Int → BigInt/Float）。

### 2.2 NaN Check（NaN 检测）

```
110     ISNAN       Is Not-a-Number
```

- **Arguments:** 单个值。
- **Return value:** Bool。
- 仅适用于 **Float type only**。等价于 `math.IsNaN(...)`。
- 当浮点值为 NaN（Not-a-Number）时返回 `true`。该值通常来自未定义的浮点运算（例如 `0.0 / 0.0`）。
- **Design note:** 提供该指令是因为 NaN 具有 `NaN != NaN` 的特性，常规相等判断无法直接识别。

### 2.3 Range Check（区间检查）

```
111     WITHIN      Range check (min <= x < max)
```

- **Arguments:**
  1. 待检查的值。
  2. 定义区间的 2 成员 slice：`[min, max]`。
- **Return value:** Bool —— 当 `min <= x < max` 时返回 `true`（半开区间：下界包含、上界不包含）。
- **Supported types:** Byte、Rune、Int、BigInt、Float。
- **Example:**
  ```
  ENV{Height} {0, 240} WITHIN
  // 检查交易所在区块高度是否位于第一天内
  // （按每天约 240 个区块估算）
  ```
- **Float caveat:** 对 Float 值进行下界相等判断（`min <= x`）时可能存在精度误差。


## 3. Type Compatibility（类型兼容性）

| Instruction | Bool | Byte | Rune | Int | BigInt | Float | String | Bytes |
|-------------|------|------|------|-----|--------|-------|--------|-------|
| EQUAL | yes | yes | yes | yes | yes | yes* | yes | yes |
| NEQUAL | yes | yes | yes | yes | yes | yes* | yes | yes |
| LT | — | yes | yes | yes | yes | yes | yes | yes |
| LTE | — | yes | yes | yes | yes | yes | yes | yes |
| GT | — | yes | yes | yes | yes | yes | yes | yes |
| GTE | — | yes | yes | yes | yes | yes | yes | yes |
| ISNAN | — | — | — | — | — | yes | — | — |
| WITHIN | — | yes | yes | yes | yes | yes* | — | — |

`*` Float 的相等/边界比较虽受支持，但可能不精确。
